# Euclidean vector space descriptor.

Euclidean vector space descriptor.

## Usage

``` r
euclidean(dim, dtype = "double")
```

## Arguments

- dim:

  Dimension of the space.

- dtype:

  Scalar type (currently only `"double"`).

## Value

An `eigencore_space` descriptor for the Euclidean space `R^dim` or
`C^dim`.
